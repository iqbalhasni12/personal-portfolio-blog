Web Architecture Recent Project

Structuring the Internlink Project with CodeIgniter 4

Managing the "Internlink" software project as Development Lead required a deep dive into structured development planning. Working heavily within the CodeIgniter 4 (CI4) framework, a major focus was remapping the development timeline to ensure steady, measurable progress tracking.

The core challenge was designing robust relational database schemas that could handle complex data flow while maintaining the strict MVC architectural patterns required by CI4. This project reinforced the importance of thorough research planning before writing a single line of backend logic.

Tech Stack: PHP, CodeIgniter 4, MySQL, HTML/CSS

Building an E-Commerce Platform from Scratch

During my Diploma in Information Technology, I took on the challenge of developing a fully functional E-Commerce website. This was my first major foray into tying frontend user interfaces directly to backend database management systems.

The project focused heavily on establishing core programming fundamentals. I had to build secure user authentication protocols, manage dynamic product inventories via database integration, and ensure the UI remained responsive and intuitive for the end-user. It was a foundational experience that directly prepared me for the more complex CI4 architectures I build today.

Tech Stack: Core Programming Fundamentals, Database Management, Web UI
